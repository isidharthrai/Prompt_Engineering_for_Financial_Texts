# Task 3: Financial Insight Generation

## Overview
This task focuses on generating actionable financial insights from financial statements using Large Language Models (LLMs) with different prompting strategies.

## Objective
Evaluate the ability of LLMs to generate meaningful, accurate, and actionable insights from financial data.
